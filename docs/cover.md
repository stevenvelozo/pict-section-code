# Pict Section Code

> A code editor and syntax highlighter for the Pict ecosystem

Wrap CodeJar in a Pict view for editable code editors and read-only syntax-highlighted displays. Built-in highlighting for JavaScript, JSON, HTML, CSS, and SQL with full support for custom highlighter functions.

- **Code Editing** -- Full-featured editor with undo, auto-indent, bracket closing
- **Read-Only Display** -- Syntax-highlighted code blocks for documentation and presentations
- **Five Languages** -- JavaScript, JSON, HTML, CSS, SQL highlighted out of the box
- **Custom Highlighters** -- Plug in Prism.js, highlight.js, or any function
- **Two-Way Binding** -- Bind code content to Pict AppData addresses
